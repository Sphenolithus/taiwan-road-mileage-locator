# 官方道路里程資料

本資料夾的 `mileposts.geojson.gz.b64` 是網站使用的正式里程點資料。

資料來源：

- 交通部公路局「省道里程坐標(里程牌標誌)」CSV，更新時間 2026-05-05。
- 交通部公路局「省道里程坐標(里程牌標誌)」CSV，另補入 data.gov.tw 2026-04-01 詮釋資料版本中的台7甲、台8、台14甲。
- 交通部高速公路局「國道百公尺里程樁」KML，含 WGS84 坐標，統計至 112 年。
- 交通部高速公路局「國2甲」CSV。
- 交通部高速公路局「國4豐潭延伸段_里程樁」CSV。

轉換後範圍：

- 高速公路：國道主線與公開資料中的國道支線。
- 快速道路：台61、台62、台64、台65、台66、台68、台72、台74、台76、台78、台82、台84、台86、台88。
- 平面道路：台1、台2、台3、台7、台7甲、台8、台9、台11、台14、台14甲、台17、台18、台20。

轉換後筆數：

- 高速公路：20,070
- 快速道路：25,006
- 平面道路：21,512
- 合計：66,588

資料品管：

- `順向`、`逆向`、`雙向` 這類非方位值已展開為各路線預設方向，避免前端只查北上/南下或東向/西向時漏掉後段里程。
- 目前已確認正式資料內只保留 `north`、`south`、`east`、`west` 四種標準方向。
- 全路線覆蓋檢核見 `coverage-report.md`。
