# 官方道路里程資料

本資料夾的 `mileposts.geojson.gz.b64` 是網站使用的正式里程點資料。

資料來源：

- 交通部公路局「省道里程坐標(里程牌標誌)」CSV，更新時間 2026-05-05。
- 交通部高速公路局「國道百公尺里程樁」KML，含 WGS84 坐標，統計至 112 年。
- 交通部高速公路局「國2甲」CSV。
- 交通部高速公路局「國4豐潭延伸段_里程樁」CSV。

轉換後範圍：

- 高速公路：國道主線與公開資料中的國道支線。
- 快速道路：台61、台62、台64、台65、台66、台68、台72、台74、台76、台78、台82、台84、台86、台88。
- 平面道路：台1、台3、台9、台17、台20。

轉換後筆數：

- 高速公路：20,070
- 快速道路：13,423
- 平面道路：8,872
